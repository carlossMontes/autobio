Hola, me llamo Carlos Alberto Montes Romero, nací el 25 de abril del 1998 a las 23:45 en Dolores Hidalgo, Guanajuato.

Actualmente tengo 22 años, estoy estudiando el ultimo cuatrimestre de la Ingeniería en TIC y trabajando desde casa.

He trabajado con distintas tecnologías de las TIC, tanto escolar como profesionalmente. Escolar y profesionalmente he trabajado en el desarrollo de software en lenguaje PHP, aprendiendo con ello HTML, CSS y muy importante -Javascript-, con y sin framework en diversos proyectos.

Actualmente estoy aprendiendo Microservicios Web con Java para backend y Angular 10 para frontend.

Con bases de datos he trabajado con varios SGBD, entre ellos SQL Server, MySQL y Oracle. Incluso, un tiempo estuve dando soporte a temas de Oracle.

El tema más constante con el que he trabajado es el análisis y desarrollo de ETLs con una suite especializada llamada Informatica PowerCenter.

Me gustaría aprender Python (con el que ya he trabajado personalmente), aprender bien Angular, Big Data y/o NoSQL (MongoDB o CassandraDB)
